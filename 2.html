CSS

Hyper Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.


This document collects together into one definition all the specs that together form the current state of Cascading Style Sheets (CSS) as of 2021.

The primary audience is CSS implementers, not CSS authors, as this definition includes modules by specification stability, not Web browser adoption rate.
